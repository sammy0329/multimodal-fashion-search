---
name: log
description: 현재 세션의 작업 내용을 DEVLOG.md에 기록
---

프로젝트 루트의 DEVLOG.md 파일에 오늘 날짜 섹션을 추가하고 현재 세션에서 수행한 작업을 기록해줘.

## 기록 형식

```markdown
## YYYY-MM-DD (마일스톤명)

### 완료한 작업
- 구체적으로 무엇을 했는지

### 변경된 파일
- 생성/수정/삭제된 파일 목록

### 트러블슈팅
- 발생한 에러와 해결 방법 (없으면 "없음")

### 기술적 결정사항
- 어떤 결정을 왜 했는지 (없으면 "없음")

### 다음 할 일
- 이어서 해야 할 작업
```

## 규칙
- DEVLOG.md가 없으면 새로 생성
- 기존 내용 위에 최신 날짜가 오도록 추가 (최신순)
- 같은 날짜가 이미 있으면 내용을 합쳐서 업데이트
- 간결하지만 나중에 봤을 때 이해할 수 있도록 구체적으로 작성
